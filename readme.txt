Universal Social Reputation enable your blog to prevent trolls from posting and auto accept comments from trusted users.

== Description ==

Warning: This plugin connect your blog with Universal Social Reputation (USR) services [This is a 3rd party service that does not belongs to Wordpress]. By using this plugin you accept sending users' data (e.g. email, comments, posts they see) to this service and accept your users' comments may be rated and accept or marked as spam those comments automatically.

Whenever a user comes to your blog, you will be able to prevent its comments. You will be able to know if he or she has been a troll in other blogs and has received a bad score previously.
If a user has got a good reputation, you will be able to act in consequence and maybe you would accept automatically her/his comment!

Please, consider warning your users that this plugin may collect data from their usage of your blog in order to know better these persons. 
This knowledge will help us to help you preventing trolls comming!

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Create an account in unisocrep.herokuapp.com and get your secret token.
4. Use the Settings->Comments' Vote screen to configure the plugin
